Getting Started

To start developing with the Casio Graph 35+E II Game Engine, follow these steps:

  1. Have a PC running Linux or Windows with WSL.
  2. Install GiteaPC (Ubuntu/Debian/WSL): 
    sudo apt install curl git python3 build-essential pkg-config
curl "https://git.planet-casio.com/Lephenixnoir/GiteaPC/raw/branch/master/install.sh" -o /tmp/giteapc-install.sh && bash /tmp/giteapc-install.sh
  3. Install Gint (Ubuntu/Debian/WSL): 
    # For WSL users:
sudo apt install patch wget cmake python3-pil libusb-1.0-0-dev libsdl2-dev libpng-dev libncurses-dev

# For Debian/Ubuntu:
sudo apt install patch wget cmake python3-pil libusb-1.0-0-dev libudisks2-dev libglib2.0-dev libpng-dev

giteapc install Lephenixnoir/gint
  4. If you are using Arch Linux or encounter problems, see:
    • GiteaPC repository
    • Gint repository
  5. Download the uncompiled version of the project:
    • Download ZIP
  6. Edit and test the code as needed.
  7. To build the project, execute at the root (where CMakeLists.txt is): 
    fxsdk build-fx
